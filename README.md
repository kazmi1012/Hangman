# Hangman 🎯

A terminal-based Hangman game built in Python.

## Features
- Random word selection
- Letter guessing with input validation
- Tracks correct and incorrect guesses
- Attempt limit with win/loss detection
- Visual feedback for remaining attempts

## How to Run

1. Clone the repo
```bash
git clone https://github.com/kazmi1012/Hangman.git
cd Hangman
```

2. Run the game
```bash
python main.py
```

## Tech Used
- Python
- Terminal I/O
- OOP (Object-Oriented Programming)
